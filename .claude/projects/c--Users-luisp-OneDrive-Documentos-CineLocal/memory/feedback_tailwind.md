---
name: Tailwind simples
description: Usuário prefere Tailwind básico e mínimo, sem classes excessivas que dificultem leitura do código
type: feedback
---

Manter Tailwind super básico — só o necessário para layout e estilo essencial.

**Why:** O usuário não consegue ler o código quando tem muito Tailwind junto (animações, transitions, hover states complexos, group states, etc).

**How to apply:** Ao escrever JSX, usar poucas classes Tailwind por elemento. Evitar transitions, animações, group-hover, rings, shadows elaborados. Priorizar legibilidade do código sobre polish visual.
